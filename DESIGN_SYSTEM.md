# ExplorerX Product Design System

ExplorerX uses one restrained consumer-product language across all screens.

- Base: near-black neutral backgrounds with three clear surface levels.
- Accent: Explorer green is reserved for primary actions and selected states.
- Type: system UI fonts, compact headings, readable body text, no display-scale hero copy.
- Radius: 10px controls, 14px panels, 18px media and modal surfaces.
- Depth: borders separate most layers; shadows are reserved for overlays and menus.
- Cards: photography leads. Content sits below the image without a surrounding dashboard box.
- Motion: short functional transitions only; reduced-motion is fully respected.
- Empty states: honest, compact and useful. Never fill missing data with artificial content.
